# PROJECTIO
Experiencia narrativa interactiva desarrollada con HTML, CSS y JavaScript.

## Descripción
PROJECTIO es una experiencia narrativa interactiva donde el jugador acompaña a Apolo, un hombre convencido de que una entidad conocida como "El Intruso" es la responsable de todas las desgracias ocurridas en su vida. A través de diálogos, decisiones y escenarios simbólicos, el jugador descubrirá una verdad que cambiará por completo la perspectiva de la historia.

## Características
- Historia interactiva con enfoque psicológico.
- Sistema de diálogos.
- Sistema de decisiones.
- Efecto de máquina de escribir.
- Transiciones y efectos visuales.
- Desarrollado únicamente con HTML, CSS y JavaScript.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript

## Ejecución
1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador web moderno.
3. Jugar PROJECTIO.
